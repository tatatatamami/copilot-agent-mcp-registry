# server-filesystem


> MCP server for secure filesystem operations with configurable directory access controls

## Overview

A Node.js MCP server implementing the Model Context Protocol for secure filesystem operations, including reading, writing, creating, listing, moving, and searching files and directories. It enforces configurable access controls via command-line arguments or the MCP Roots protocol, restricting all operations to explicitly allowed directories.
